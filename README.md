# 🧠 Mind Marathon

> A modern JavaFX-based quiz game packed with categories, achievements, lives, coins, hints, and competitive progression mechanics.

## 🎮 Features

- 🧩 Multiple quiz categories
- ⚡ Easy, Medium & Epic difficulty modes
- ⏳ Timed quiz challenges
- 💰 Coins & rewards system
- ❤️ Lives regeneration system
- 💡 Hint purchasing system
- 🏆 Achievements & medals
- 📊 Game history tracking
- 👤 User authentication & profiles
- 🛍️ In-game shop system
- 🎨 Modern JavaFX UI

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Jannat-ul-Mava/Mind_marathon_game.git
cd Mind_marathon_game
```

### Run Project

Add JavaFX SDK and run:

```bash
--module-path "path/to/javafx/lib" --add-modules javafx.controls,javafx.fxml
```

Run:

```bash
Welcome_page.java
```
